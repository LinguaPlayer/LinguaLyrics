# Lingua Lyrics

A lyric app that works with most of Linux music players.(Tested with clementine)

![screenshot](./LinguaRepo/assets/screenshot.png)

### dependencies:
Python3

pip3 install -r requirements.txt

### How to run:
`python3 ./start.py`
