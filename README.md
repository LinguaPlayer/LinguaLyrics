# Lingua Lyrics

A lyric app that works with most of Linux music players.

# UNDER HEAVY DEVELOPMENT
Lingualyrics is still under heavy development but you can try it with below instructions :)

### dependencies:
Python3

pip3 install -r requirements.txt

### How to run:
`python3 ./start.py`
